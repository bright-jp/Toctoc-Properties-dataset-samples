# Toctoc Properties Listings Dataset Sample

<h2>1,003件のレコードを含むサンプルデータセット</h2>

<a href="https://brightdata.jp">
    <img src="https://github.com/bright-jp/Toctoc-Properties-dataset-samples/blob/main/datasets-image.png" alt="Bright Data datasets" />

この**Toctoc Properties Listings Dataset Sample**には**1,003件のレコード**が含まれており、**Bright Data API**を使用して抽出されました。

## データポイント

| Field Name | Description |
|------------|-------------|
| `Imagen` | 物件のさまざまな外観や特徴を示す画像のURL。 |
| `No_de_imagenes` | 物件リスティングで利用可能な画像の総数。 |
| `Descripcion` | 特徴や設備を含む物件の詳細説明。 |
| `Precio` | 現地通貨または単位で表示された物件価格。 |
| `Currency` | UFやCLPなど、物件価格が表示される通貨の種類。 |
| `Ubicacion` | 物件の地理座標（緯度・経度）。 |
| `Habitaciones` | 物件で利用可能なベッドルーム数。 |
| `Banos` | 物件で利用可能なバスルーム数。 |
| `Dimension_propiedad` | 物件の面積（平方メートル）。 |
| `Dimension_terreno` | 物件が所在する土地の面積（平方メートル）。 |
| `Titulo` | 物件リスティングのタイトルまたは見出し。 |
| `Phone1` | 問い合わせ用の主連絡先電話番号。 |
| `Phone2` | 問い合わせ用の副連絡先電話番号。 |
| `Email` | 物件の売主またはエージェントに連絡するためのメールアドレス。 |
| `Seller` | 物件を掲載している売主または不動産エージェントの名前。 |
| `Tipo_de_vendedor` | 売主の種類（個人または代理店）。 |
| `Type` | 物件の種類（戸建て、アパートなど）。 |
| `Nueva_usada` | 物件が新築か中古かを示します。 |
| `Venta_Arriendo` | 物件が売買用か賃貸用かを示します。 |
| `Region` | 物件が所在する地理的地域。 |
| `Comuna_Ciudad` | 物件所在地の具体的な都市またはコミューン。 |
| `Year` | 物件の建築年または最終改装年。 |
| `Parqueaderos` | 物件に付帯する駐車スペース数。 |
| `ID` | 物件リスティングの一意識別子。 |
| `url` | オンライン物件リスティングへのWebリンク。 |
| `seller_id` | 売主またはエージェントの一意識別子。 |
| `whatsapp_number` | 直接連絡のためのWhatsApp連絡先番号。 |


## ユースケース

### 1. 不動産市場分析
このデータセットを活用して、地域ごとの物件価格や供給状況の市場トレンドを分析し、戦略的な意思決定に役立てます。

### 2. 投資機会の特定
物件の特徴や価格動向を調査することで、潜在的な投資機会を特定し、十分な情報に基づいた投資判断を行います。

### 3. 不動産会社向け競合分析
物件リスティング、売主タイプ、価格戦略を評価して競合分析を実施し、ビジネス戦略の改善に役立てます。
