# Toctoc Properties Listings Dataset Sample

<h2>1,003件のレコードを含むサンプルデータセット</h2>

この **Toctoc Properties Listings Dataset Sample** には **1,003件のレコード** が含まれており、**Bright Data API** を使用して抽出されました。

## Data Points

| Field Name | Description |
|------------|-------------|
| `Imagen` | さまざまな視点や特徴を示す物件画像のURLです。 |
| `No_de_imagenes` | 物件リスティングで利用可能な画像の総数です。 |
| `Descripcion` | 特徴や設備を含む、物件の詳細説明です。 |
| `Precio` | 現地通貨または単位で表示された物件価格です。 |
| `Currency` | UFやCLPなど、物件価格が表示される通貨タイプです。 |
| `Ubicacion` | 物件の地理座標（緯度・経度）です。 |
| `Habitaciones` | 物件のベッドルーム数です。 |
| `Banos` | 物件のバスルーム数です。 |
| `Dimension_propiedad` | 物件の面積（平方メートル）です。 |
| `Dimension_terreno` | 物件が所在する土地の面積（平方メートル）です。 |
| `Titulo` | 物件リスティングのタイトルまたは見出しです。 |
| `Phone1` | 問い合わせ用の主要連絡先電話番号です。 |
| `Phone2` | 問い合わせ用の副次連絡先電話番号です。 |
| `Email` | 物件の売主またはエージェントへの連絡用メールアドレスです。 |
| `Seller` | 物件を掲載している売主または不動産エージェント名です。 |
| `Tipo_de_vendedor` | 個人または代理店など、売主のタイプです。 |
| `Type` | 戸建て、アパートなどの物件タイプです。 |
| `Nueva_usada` | 物件が新築か中古かを示します。 |
| `Venta_Arriendo` | 売買か賃貸かを指定します。 |
| `Region` | 物件が所在する地理的な地域です。 |
| `Comuna_Ciudad` | 物件所在地の特定の都市またはコミューンです。 |
| `Year` | 物件の建築年または最終改修年です。 |
| `Parqueaderos` | 物件に付帯する駐車スペース数です。 |
| `ID` | 物件リスティングの一意識別子です。 |
| `url` | オンライン物件リスティングへのWebリンクです。 |
| `seller_id` | 売主またはエージェントの一意識別子です。 |
| `whatsapp_number` | 直接連絡のためのWhatsApp連絡先番号です。 |


## Use Cases

### 1. Real Estate Market Analysis
このデータセットを活用して、地域ごとの物件価格や供給状況の市場トレンドを分析し、戦略的意思決定に役立てます。

### 2. Investment Opportunities Identification
物件の特徴や価格動向を検討して潜在的な投資機会を特定し、情報に基づいた投資判断を行います。

### 3. Competitor Analysis for Real Estate Agencies
物件リスティング、売主タイプ、価格戦略を評価して競合分析を行い、事業戦略の改善につなげます。
